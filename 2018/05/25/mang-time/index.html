时间管理

发表于 2018-05-25 | 分类于 自学 |
  • 做车的时候和排队的时候轻听APP听力练习，听英文读物
  • 中午或晚上找时间看书，坚持每天看，看完一本写出200-300字的短评
  • 每天用半小时左右做笔记或思维导图
  • 每周总结学习进度，进行反思并进行相应调整
  • 每三、四个月或半年做次主题阅读，选择一个目标集中阅读10、20本相关书籍，做笔记、思维导图，进行汇总，形成知识框架。把合适的内容输出到BLOG
  • 1-2年安排时间参加合适的培训
  • 准备考一些有价值的证书，行业内的顶级证书
  • 每天拿出时间尽量完成一课学习，罗得塔石碑或全球说，一般20分钟以内
